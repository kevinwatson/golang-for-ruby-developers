### Chapter 10 - Logging

## Introduction

Ruby on Rails supports log levels. This allows you to write logging details in different modes. For example, you can add write `debug` lines which only print when the app is configured in `debug` mode, or when you're running the application locally. These debug lines can be verbose because they add no overhead to the app when it's running in your production environment.
